# Demandas de muestra

PDFs de demandas de muestra para desarrollo y demo. Al menos una demanda de
**tránsito** realista (redactada por los abogados) es el caso guía del proyecto
(ver PLAN.md: "un tipo de extremo a extremo antes de generalizar").

## Convenciones

- Formato: PDF (lo procesa `src/extraction.py` con PyMuPDF).
- **No subir datos personales reales.** Usar demandas ficticias o anonimizadas.
- Nombrar por tipo de hecho, p. ej. `transito_01.pdf`.

> Pendiente de aportar por los abogados.
